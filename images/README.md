Asset folders are grouped by where they are used in the SoftEx site.

- `archive/`: unused backup files kept for reference
- `audience/`: "Who Should Join" icons
- `backgrounds/`: shared decorative backgrounds
- `branding/`: favicon, webclip, and brand marks
- `hero/`: homepage hero graphics and background
- `icons/`: reusable UI and section icons
- `people/`: advisor and director portraits, with `mobile/` variants nested by role
- `program/`: tuition graphics and schedule reference assets
- `site-visits/`: exclusive site visit photography
- `speakers/`: speaker collages and speaker-section backgrounds
- `trips/`: experiential learning trip artwork
